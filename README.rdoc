= LightweightLocale Plugin

A lightweight locale plugin for Ruby on Rails.

- It sets the <code>I18n.locale</code> based on a query param called +locale+ or the HTTP header.

- It stores the locale in the session.

- It provides the +select_locale_tag+ helper.
